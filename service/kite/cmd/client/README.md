# cani.kite.client

Kite client for the kickable API.

## Usage

```shell
./bin/cani.kite.client -kick "it" -addr http://127.0.0.1:3000/kite
```

| Flag | Default | Description |
|------|---------|-------------|
| `-kick` | `""` | The item to determine as kickable |
| `-addr` | `http://127.0.0.1:3000/kite` | Kite server URL |
