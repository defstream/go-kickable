# cani.http.client

HTTP client for the kickable API.

## Usage

```shell
./bin/cani.http.client -kick "it" -address http://localhost:3000
```

| Flag | Default | Description |
|------|---------|-------------|
| `-kick` | `""` | The item to determine as kickable |
| `-address` | `http://localhost:3000` | Server address |
