# cani.gob.rpc.client

GOB RPC client for the kickable API.

## Usage

```shell
./bin/cani.gob.rpc.client -kick "it" -address localhost:4000
```

| Flag | Default | Description |
|------|---------|-------------|
| `-kick` | `""` | The item to determine as kickable |
| `-address` | `localhost:4000` | Server address |
