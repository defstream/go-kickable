# cani.json.rpc.client

JSON-RPC client for the kickable API.

## Usage

```shell
./bin/cani.json.rpc.client -kick "it" -address :4000
```

| Flag | Default | Description |
|------|---------|-------------|
| `-kick` | `""` | The item to determine as kickable |
| `-address` | `:4000` | Server address |
