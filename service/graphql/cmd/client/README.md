# cani.graphql.client

HTTP client that queries the GraphQL kickable service.

## Usage

```shell
./bin/cani.graphql.client -kick "it" -address http://localhost:8000
```

| Flag | Default | Description |
|------|---------|-------------|
| `-kick` | `""` | The item to determine as kickable |
| `-address` | `http://localhost:3000` | Server address |
